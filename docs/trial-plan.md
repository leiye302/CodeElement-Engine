# Trial Plan

码元引擎计划开放基础 API 试用能力，用于收集真实反馈、验证多场景表达效果，并为后续框架授权和定制化部署建立早期用户基础。

## Trial Goals

- Validate SceneCode expression in different embodied-intelligence scenarios.
- Collect developer and research feedback.
- Test task-planning outputs under different input formats.
- Identify reusable skill templates from successful workflows.
- Prepare future API charging and framework-licensing models.

## Target Trial Users

| User Type | Trial Value |
| --- | --- |
| University labs | Rapidly validate robotic task flows and scene expression. |
| Research institutions | Test new algorithms and embodied-intelligence workflows. |
| Developers | Explore perception, decision, and task-planning interfaces. |
| System integrators | Evaluate cross-device reuse and deployment cost reduction. |
| Manufacturing companies | Try lightweight intelligent-upgrade scenarios. |

## Planned Trial Capabilities

- Upload image, point cloud, or task metadata.
- Return SceneCode-style scene expression.
- Return task-level decision output.
- Provide workflow feedback template.
- Provide sample scenarios and example payloads.
